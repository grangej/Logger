Logger

Logger is a extendable logging platform supporting various severity levels.

About

Logger is used to send logs to both console and remote / crash / collection frameworks.

The framework consits of the main Logger enum and Extensions that add support for addtional reporters. The reports support the ability to inject frameworks that are not part of the logger platform to prevent any dependnices. Currently we still need to compile in addtional collectors.
